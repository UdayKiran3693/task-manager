# Team Task Manager

## Live Links
Frontend: https://task-manager-peach-nu.vercel.app/
Backend: task-manager-production-c601.up.railway.app

## Features
- User Authentication
- Create Tasks
- Task Status Update
- Dashboard
- MongoDB Database
- REST APIs
- Role-based structure

## Tech Stack
Frontend:
- React
- Axios
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Installation

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
